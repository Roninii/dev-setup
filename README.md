### Install Homebrew
```bash
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```

### Install the homebrew tap
```bash
  brew tap Homebrew/bundle
```

### Create a brewfile with all Homebrew packages installed
```bash
  brew tap Homebrew/bundle
```

### Restore config
- clone this repo
```bash
  cd config
  brew bundle
```
